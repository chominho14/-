입출력 예
예를들어 str이 "1234"이면 1234를 반환하고, "-1234"이면 -1234를 반환하면 됩니다.
str은 부호(+,-)와 숫자로만 구성되어 있고, 잘못된 값이 입력되는 경우는 없습니다.

풀이과정.

1. -부호인 경우를 위해 s를 쪼개주고 -를 뺀 숫자를 기억하는 변수를 만들어 저장했다.
2. if문을 돌려 -가 붙어있으면 -1을 곱해서 출력 아니면 그냥 출력.


코드

function solution(s) {
    const spStr = s.split("");
    const slStr = s.slice(1,s.length);
    
    if(spStr[0] === "-"){
        return Number(slStr) * -1;
    } else {
        return Number(s);
    }
}
