
문제 설명
문자열 s에 나타나는 문자를 큰것부터 작은 순으로 정렬해 새로운 문자열을 리턴하는 함수,
 solution을 완성해주세요.
s는 영문 대소문자로만 구성되어 있으며, 대문자는 소문자보다 작은 것으로 간주합니다.

풀이과정

1. split으로 쪼개준다.
2. sort를 이용해 오름차순으로 정렬한다.
3. reverse로 뒤집어 준다.
4. join을 이용해 함쳐준다.

코드

function solution(s) {
    return s.split("").sort().reverse().join("")
}
