 *으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

풀이과정

1. repeat을 이용하여 문자열을 어느 갯수만큼 반복한다.
2. substr을 이용하여 문자열을 마지막에서 4개만 잘라준다.

* 문자열을 반복하는 2가지 방법.
repeat으로 반복, array(n+1).join("문자열")로 반복한다.
ex) Array(5).join("a"); 
-> "aaaa" 반환


코드

function solution(phone_number) {
    let result = "";
    return result = "*".repeat(phone_number.length-4) + phone_number.substr(phone_number.length - 4 , 4);
}


다른풀이

function hide_numbers(s) {
  return s.replace(/\d(?=\d{4})/g, "*");
}
